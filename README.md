# Rag_study

## 소개

이 저장소는 Retrieval-Augmented Generation (RAG), Agent, GraphRAG, 배포까지 단계적으로 학습하고 기록하는 스터디 공간입니다.

## 학습 폴더 구조

- 01-Foundations
- 02-Agent-Design
- 03-Orchestration-Layer
- 04-Context-Memory
- 05-MCP
- 06-Agent-Governance
- 07-Key-Capabilities
- 08-Retrieval-Augmented-Generation
- 09-Application-Patterns
- 10-Advanced-Agents
- 11-Graph-Enhanced-RAG
- 12-Production-Deployment
- 13-RAG-Paper-Study

각 폴더에는 주제별 정리용 `README.md`가 있으며, 아래 항목 중심으로 기록합니다.

- 학습 목표
- 정리 노트 (날짜/키워드/배운 점/막혔던 점/다음 액션)
- 참고 자료

## 논문 정리 방식

논문 학습은 `13-RAG-Paper-Study` 폴더에서 관리합니다.

- `논문_정리_인덱스.md`: 읽은 RAG 관련 논문 목록(날짜, 한줄 요약, 링크, 정리 파일)
- `논문_요약_템플릿.md`: RAG 논문 전용 정리 템플릿

권장 흐름:

1. 논문 1편 읽기
2. `논문_요약_템플릿.md` 복사해 새 파일 생성
3. `논문_정리_인덱스.md`에 링크 추가
